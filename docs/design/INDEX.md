## 使用建议

- 按功能实现顺序回顾项目时，优先阅读 `features/` 下的文档。
- 查看体验、交互、展示层优化时，优先阅读 `optimizations/` 下的文档。
- 后续新增功能模块时，继续在 `features/` 下按顺序编号追加。
- 后续新增体验类或重构类设计时，继续在 `optimizations/` 下按顺序编号追加。
