# 评论模块测试

## 1. 未登录不能评论
### 前置条件
- 当前未登录
- 已存在文章

### 步骤
1. 进入文章详情页
2. 输入评论内容
3. 提交评论

### 预期结果
- 提交失败
- 提示先登录
- 评论列表无新增内容

## 2. 登录后发表评论成功
### 前置条件
- 当前已登录
- 已存在文章

### 步骤
1. 进入文章详情页
2. 输入评论内容
3. 提交评论

### 预期结果
- 评论成功
- 评论列表出现新评论
- 文章评论数增加

## 3. 评论列表展示正确
### 前置条件
- 已存在评论

### 步骤
1. 打开文章详情页

### 预期结果
- 可看到评论作者
- 可看到评论正文
- 评论列表顺序正常
