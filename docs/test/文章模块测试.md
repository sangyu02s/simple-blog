# 文章模块测试

## 1. 未登录不能发帖
### 前置条件
- 当前未登录

### 步骤
1. 打开发帖页
2. 输入标题、摘要、标签、正文
3. 提交表单

### 预期结果
- 提交失败
- 提示先登录

## 2. 登录后发帖成功
### 前置条件
- 当前已登录

### 步骤
1. 打开发帖页
2. 输入有效标题、摘要、标签、正文
3. 提交表单

### 预期结果
- 发帖成功
- 跳转文章详情页

## 3. 首页显示文章列表
### 前置条件
- 至少存在一篇文章

### 步骤
1. 打开首页

### 预期结果
- 显示文章列表
- 可看到标题、摘要、作者、标签、点赞数、评论数

## 4. 文章详情展示正确
### 前置条件
- 至少存在一篇文章

### 步骤
1. 在首页点击文章标题

### 预期结果
- 进入文章详情页
- 显示标题、摘要、正文、作者、标签、点赞数、评论数、浏览数
