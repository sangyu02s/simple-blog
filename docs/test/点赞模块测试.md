# 点赞模块测试

## 1. 未登录不能点赞
### 前置条件
- 当前未登录
- 已存在文章

### 步骤
1. 进入文章详情页
2. 点击点赞按钮

### 预期结果
- 点赞失败
- 提示先登录
- 点赞数不变

## 2. 登录后点赞成功
### 前置条件
- 当前已登录
- 当前用户尚未点赞该文章

### 步骤
1. 进入文章详情页
2. 点击点赞按钮

### 预期结果
- 点赞成功
- 按钮切换为取消点赞
- 点赞数增加

## 3. 取消点赞成功
### 前置条件
- 当前已登录
- 当前用户已点赞该文章

### 步骤
1. 点击取消点赞按钮

### 预期结果
- 取消点赞成功
- 按钮切换为点赞
- 点赞数减少

## 4. 点赞状态展示正确
### 前置条件
- 已存在文章

### 步骤
1. 登录后进入文章详情页

### 预期结果
- 按钮状态与当前是否已点赞一致
- 页面显示的点赞数与接口数据一致
